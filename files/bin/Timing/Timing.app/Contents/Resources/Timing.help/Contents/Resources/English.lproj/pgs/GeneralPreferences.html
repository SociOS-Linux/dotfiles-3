General Preferences

Timing offers you a few options to customize its behavior:

  • Project "Unassigned" - Include applications with a project: this option is no longer available.
  • Helper Application - Launch automatically at login: Determines whether the Timing menu item should be started as soon as you log in.
  • Helper Application - Quit when exiting Timing: Determines whether the Timing menu item should be quit as soon as you exit the main Timing application. It's recommended to leave this unchecked, because otherwise tracking will stop as soon as you quit Timing.
  • Helper Application - Suspend tracking after being idle for at least XX seconds: If checked, Timing will suspend tracking if you haven't used your Mac in the past XX seconds. Tracking will resume automatically as soon as you start using your Mac again.
  • Helper Application - Maximum URL tracking level: Sets the maximum number of URL components that Timing's going to track. The default value is 1 - only the domain of visited websites will be tracked. If you increase this value, Timing's database will grow quicker and the application might become slower. Setting this to zero disables website-level tracking completely (only time for Safari or Chrome etc. will be tracked), setting it to -1 causes Timing to always track the entire URL.
